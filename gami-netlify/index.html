⚙️ O que é o N8N?
É o "cérebro" que conecta tudo. Quando algo acontece (nova mensagem, novo lead, nova data) o N8N executa ações automáticas em outros sistemas.
📦 Opção 1 — N8N Cloud (recomendado)
Passo 1 — Criar conta
Acesse n8n.io/cloud → clique em "Start for free" → crie sua conta com email
Passo 2 — Escolher o plano
Plano Starter (~$24/mês) é suficiente para começar. Inclui 5.000 execuções/mês, mais que suficiente.
✓ Pronto para usar em 5 minutos
🖥️ Opção 2 — N8N Self-hosted (gratuito)
Se tiver um servidor (VPS na DigitalOcean ou Railway), rode grátis com Docker:

docker run -it --rm --name n8n -p 5678:5678 -v n8n_data:/home/node/.n8n docker.n8n.io/n8nio/n8n
Acesse em: http://seuip:5678
🔑 Conectar Claude ao N8N
1. No N8N, clique em "Credentials" → "New Credential"
2. Busque por "Anthropic" → selecione
3. Cole sua API key do Claude (console.anthropic.com → API Keys)
4. Salve — pronto! Agora pode usar o nó "Claude" em qualquer workflow
💬 WhatsApp Business API
Permite enviar mensagens automáticas para gestores da Gâmi, clientes da Jipa e grupos internos.
🛤️ Opção 1 — Meta Business (oficial)
1. Acesse developers.facebook.com → crie um app
2. Adicione o produto "WhatsApp" ao app
3. Configure um número de telefone dedicado para a API
4. Crie os templates de mensagem (ex: "Boa semana, gestor de [cidade]!")
5. Aguarde aprovação dos templates (~24h)
6. No N8N, adicione o nó "WhatsApp Business Cloud" com seu token
Custo: ~R$ 0,08-0,30 por mensagem. Primeiras 1.000/mês grátis.
⚡ Opção 2 — Evolution API (gratuito, mais fácil)
⚠️ Atenção: Não é oficial. Funciona conectando pelo QR Code. Risco de ban se usado em excesso.
1. Instale no Railway ou VPS: github.com/EvolutionAPI/evolution-api
2. Leia o QR Code com o WhatsApp do número
3. Use a API REST para enviar mensagens via N8N
Custo: Gratuito (só paga o servidor ~$5/mês)
📸 Instagram Graph API
Leia DMs, comente em posts e obtenha insights dos 10 perfis da Gâmi e dos clientes da Jipa.
🔧 Como conectar os 10 Instagrams da Gâmi
1. Cada perfil precisa ser conta comercial → Configurações do Instagram → Conta → Mudar para conta profissional
2. Conecte cada Instagram a uma Página do Facebook no Meta Business Suite
3. Acesse developers.facebook.com → seu app → adicione "Instagram Basic Display"
4. Gere tokens de acesso para cada conta (você já tem o token do Meta Ads — use o mesmo app)
5. No N8N, adicione o nó "Instagram" com o token de cada perfil
